# StyleSync 👗✨

A Pinterest-inspired fashion website where users can explore styles and save their favorite looks.

## 🔗 Live Demo
https://Exuberant-649.github.io/StyleSync

## 💻 Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- LocalStorage (JSON)

## ✨ Features
- Save / Unsave outfits ❤️
- Dark / Light mode 🌙
- Search bar 🔍
- Multiple fashion categories
